# AI Guide

## Goal

This file is the project-adopted AI guide copied from the Dorado mother spec. AI must read the project-adopted rules first instead of improvising from the mother spec again.

## Working Order

1. Read `.skillrc`
2. Read `SKILL.index.json`
3. Read the project-adopted conventions under `docs/project/`
4. Read the relevant `SKILL.md`
5. Read the active change files

## Required Behavior

- Locate context through the index before opening code
- Read project-adopted conventions before implementation
- Update `SKILL.md` when code behavior changes
- Rebuild `SKILL.index.json` when needed
